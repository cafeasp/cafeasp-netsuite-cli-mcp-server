# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-04-04

### Added
- **@cafeasp/netsuite-core**: TBA authentication with HMAC-SHA256 signing
- **@cafeasp/netsuite-core**: NetSuite REST API client (CRUD operations)
- **@cafeasp/netsuite-core**: SuiteQL query engine with auto-pagination
- **@cafeasp/netsuite-core**: Config management with named profiles and env var overrides
- **@cafeasp/netsuite-cli**: Interactive config setup wizard
- **@cafeasp/netsuite-cli**: `query` command for SuiteQL with table/JSON/CSV output
- **@cafeasp/netsuite-cli**: `record` commands for CRUD operations
- **@cafeasp/netsuite-cli**: `info` command for connectivity testing
- **@cafeasp/netsuite-mcp**: Read-only MCP server with 5 tools
- **@cafeasp/netsuite-mcp**: SuiteQL query tool
- **@cafeasp/netsuite-mcp**: Record get/list tools
- **@cafeasp/netsuite-mcp**: Metadata browsing tools (list types, describe record)
